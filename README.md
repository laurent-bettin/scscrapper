1) Requirement
--------------

* Python 3.3
* [Beautiful soup][1]
* [Requests][2]
* [html5lib][3]

validate.py require [validictory][4]

2) Usage
--------

```
python path/to/scscrapper.py Username
```

3) Output
---------

A file named sc-collection.json in same folder than script.
